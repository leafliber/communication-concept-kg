### 英文全称

- GPRS Tunneling Protocol

### 中文释义

- GPRS 隧道协议

### 解释

GTP 是 3GPP 定义的核心网内部和核心网与无线网之间的数据传输协议，主要用于 4G LTE/EPC 网络中的隧道建立和管理。GTP 在 5G NSA 场景下仍然使用，而 5G SA 核心网内部改用 HTTP/2 ([[SBI]]) 接口。

### GTP 版本

| 版本 | 说明 | 应用 |
|------|------|------|
| **GTP-U** | User Plane Tunneling Protocol | 用户面数据传输 (数据平面) |
| **GTP-C** | Control Plane Tunneling Protocol | 控制面信令 (隧道管理) |
| **GTP'** | GTP Prime | 计费和流量监控 |

### GTP 在 LTE/EPC 中的位置

```
┌─────────────────────────────────────────────────────────────┐
│                      LTE/EPC 核心网                         │

│         │                │                                  │
│         │ S1-C          │ S5/S8-C                           │
│         ↓                ↓                                  │
│      MME              PCRF                                 │
│         │                                                      │
│         │ S11                                                           │
│         ↓                                                               │
│      MME ←───── GTP-C 控制面隧道建立/删除                         │
│                                                              │
│  ┌─────────────────────────────────────────────────────────┐ │
│  │              GTP-U 用户面数据隧道                         │ │

```

### GTP 隧道建立流程

| 步骤 | 说明 |
|------|------|
| 1 | UE 发起 Attach 请求 |
| 2 | MME 通过 [[GTP-C]] 与 S-GW 建立会话隧道 |
| 3 | MME 通过 GTP-C 与 P-GW 建立会话隧道 |
| 4 | [[GTP-U]] 隧道建立，承载用户数据 |
| 5 | UE 通过 [[EPS Bearer]] 传输数据 |

### GTP vs 5GC 接口协议

| 接口 | LTE/EPC 协议 | 5GC/SA 协议 |
|------|-------------|-------------|
| **核心网内部** | GTP (GTP-C/GTP-U) | HTTP/2 ([[SBI]]) |
| **无线-核心网** | S1-U (GTP-U) | NG-U ([[UPF]]) |
| **会话管理** | GTP-C | HTTP/2 (Nsmf) |
| **计费** | GTP' | Diameter / HTTP |

### GTP-U 隧道特点

| 特性 | 说明 |
|------|------|
| **端到端隧道** | UE 到 P-GW 建立独立隧道 |
| **TEID 标识** | 每个隧道通过 Tunnel Endpoint ID 唯一标识 |
| **用户面转发** | 负责所有用户 IP 数据包的转发 |
| **QoS 保障** | 配合 [[QCI]] 实现差异化 [[QoS]] |

### NSA 场景下的 GTP

在 [[NSA]] 组网下，控制面仍走 [[EPC]]，使用 GTP：

| 承载类型 | 协议 |
|---------|------|
| **4G 锚点承载** | GTP (EPC) |
| **5G 数据增强承载** | [[NR]] 空口 + [[5GC]] 用户面 |

### 相关概念

- [[EPC]] - GTP 的主要应用网络
- [[MME]] - 通过 GTP-C 控制隧道
- [[S-GW]] / [[P-GW]] - GTP 隧道的端点
- [[EPS Bearer]] - GTP 隧道承载的业务
- [[SBI]] - 5GC 用 HTTP/2 替代 GTP-C

> 反向链接: [[NSA]] 场景下，4G 锚点使用 [[GTP]] 隧道，而 5G 数据面走 [[NR]] 空口，形成双连接架构
