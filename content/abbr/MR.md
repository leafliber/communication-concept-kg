### 英文全称

- Measurement Report

### 中文释义

- 测量报告

### 解释

1. **定义与产生机制**：  
    MR 是**终端**（UE，如手机）在连接状态下，根据基站（eNodeB/gNB）下发的测量配置，对服务小区和邻区的信号质量进行测量后，上报给基站的**原始数据记录**。
    
    - **触发方式**：
        - **周期性上报**：每隔固定时间（如 480ms, 1s）上报一次，用于持续监控。
        - **事件触发上报**：当信号满足特定条件时上报（例如：邻区信号比服务小区好一定门限，触发切换）。
    - **承载内容**：MR 数据包中包含了丰富的无线环境信息。
2. **核心数据字段** (MR 里有什么？)  
    一条典型的 MR 记录通常包含以下关键信息：
    
    - **服务小区信息**：Cell ID (PCI/ECGI), 服务小区的信号强度 (**RSRP**) 和质量 (**RSRQ/SINR**)。
    - **邻区信息**：最多可包含 6-8 个最强邻区的 Cell ID 及其对应的 RSRP/RSRQ。
    - **时间戳**：精确到毫秒的测量时间。
    - **位置信息**：
        - **传统 MR**：只有小区 ID（粗略位置，几百米到几公里范围）。
        - **定位 MR** (MR-LTE/NR)：如果终端支持 GNSS 或基站支持 OTDOA/RTT 定位，MR 中可能包含经纬度坐标（精确到米级）。
    - **TA** (Timing Advance)：时间提前量，可用于估算终端距离基站的远近。
    - **PHR** (Power Headroom Report)：功率余量，反映终端发射功率是否受限。
3. **MR 的核心应用场景**：  
    MR 数据是无线网络优化（RF Optimization）的“金矿”，主要用于：
    
    - **覆盖评估**：通过海量 MR 的 RSRP 分布，绘制全网的**覆盖热力图**，识别弱覆盖区（黑洞）和过覆盖区。
    - **干扰分析**：通过 SINR（信噪比）和 RSRQ 数据，定位高干扰区域（如模三干扰、外部干扰）。
    - **切换优化**：分析切换成功率，识别“乒乓切换”（频繁来回切换）、“切换失败”或“晚切换/早切换”问题，优化邻区列表和切换参数。
    - **自动邻区关系** (ANR)：基站利用 UE 上报的未知邻区 PCI，自动发现并添加邻区关系，减少人工配置错误。
    - **精准规划**：结合 MR 中的 TA 和角度信息，辅助基站选址和天线倾角调整。
4. **MR 数据的处理流程** (从空口到大数据)  
    MR 数据量极大（每用户每秒可能产生多条），其处理链路如下：
    
    1. **UE -> NE** (基站)：UE 通过 RRC 信令将 MR 发送给基站。
    2. **NE -> EMS/OMC**：基站将 MR 进行初步统计或直接透传，上报给 OMC-R (无线网管)。
    3. **OMC -> NMS/OSS** (MDAS)：
        - 在现代网络中，OMC 通常不会存储所有原始 MR（数据量太大）。
        - 原始 MR 会被实时流式传输到**大数据平台**或**MDAS** (Mobile Data Analytics System)。
    4. **清洗与关联**：在大数据平台中，MR 数据会与话单 (CDR)、告警、工参（基站经纬度、方位角）进行关联清洗。
    5. **可视化与应用**：生成栅格化地图（Grid Map），供网优工程师使用。
5. **MR 与其他概念的关系**：
    
    - **MR vs PM**：
        - **PM** (性能管理) 是**统计值**（KPI），例如“某小区平均下载速率”、“某小区切换成功率”。它是宏观的、聚合的。
        - **MR** 是**样本值**（Raw Data），是微观的、基于单个用户单次测量的。PM 指标往往是由海量 MR 数据统计计算出来的。
    - **MR vs CDT** (Call Detail Trace)：
        - CDT 记录了呼叫建立过程中的完整信令流程。
        - MR 专注于无线信号环境的物理层测量。两者常结合使用进行深度故障定界。
6. **发展趋势**：
    
    - **MR 定位** (MR-Based Positioning)：在没有 GPS 的室内场景，利用海量 MR 的指纹库进行高精度定位（可达 10-20 米精度），用于室内导航或紧急救援。
    - **AI 赋能**：利用机器学习算法分析历史 MR 数据，预测网络拥塞趋势，实现“预测性优化”。
    - **5G NR MR**：5G 引入了更丰富的波束测量（Beam Measurement），MR 中不仅包含小区级信号，还包含**波束级** (Beam-level) 的信号质量，这对毫米波和高频段的优化至关重要。
