### 什么是室分深度覆盖

室内分布系统（室分）深度覆盖是指针对写字楼、商场、地铁、地下车库等室内场景，专门设计部署的无线信号覆盖方案，解决室内信号弱、容量不足的问题。

### 室内覆盖难点

| 场景 | 问题 | 影响 |
|------|------|------|
| **高层楼宇** | 窗外信号强但切换频繁 | 掉话、上网慢 |
| **地下室/电梯** | 信号无法穿透 | 脱网、无法通话 |
| **大型商场** | 用户密集、容量不足 | 卡顿、拥塞 |
| **地铁隧道** | 隧道结构特殊 | 切换困难、覆盖不连续 |

### 室内覆盖解决方案

| 方案 | 适用场景 | 特点 |
|------|---------|------|
| **[[宏站]]覆盖** | 室内浅层覆盖 | 依靠窗外信号穿透 |
| **室外对打** | 中小型室内 | [[楼间对打]] 方案 |
| **传统室分** | 大型楼宇、商场 | 同轴电缆 + 功分器 |
| **新型数字室分** | 高端楼宇、机场 | [[pRRU]] + 头端 |
| **[[微站]]** | 容量补充 | Small Cell |
| **[[AAU]]** | 室外覆盖室内 | 华为 Book RRU |

### 传统室分 vs 新型数字室分

| 特性 | 传统室分 (DAS) | 新型数字室分 |
|------|--------------|-------------|
| **布线** | 同轴电缆 | 光纤/网线 |
| **MIMO 支持** | 困难（需双缆） | 原生支持 |
| **[[Massive MIMO]]** | 不支持 | 可支持 |
| **维护** | 故障排查困难 | 可远程监控 |
| **[[KPI]] 可视性** | 差 | 好 |
| **部署成本** | 低（大型场景） | 高（小型场景） |
| **典型设备** | [[pRRU]] | 数字分布式 |

### 室内覆盖关键指标

| 指标 | 要求 | 说明 |
|------|------|------|
| **[[RSRP]]** | ≥ -110 dBm | 基本覆盖要求 |
| **[[SINR]]** | ≥ 0 dB | 基本质量要求 |
| **参考信号覆盖** | ≥ -105 dBm | 室内各点 |
| **切换次数** | 最小化 | 减少室内外切换 |

### 室分深度覆盖规划要点

| 要点 | 说明 |
|------|------|
| **切换带设计** | 室内外切换区域合理设置 |
| **MIMO 配置** | 充分利用多天线提升容量 |
| **容量预留** | 热点区域按峰值用户规划 |
| **[[干扰分析]]** | 避免与室外宏站干扰 |
| **[[时钟同步]]** | TDD 室分需严格时钟同步 |

### 典型室分场景方案

| 场景 | 推荐方案 | 说明 |
|------|---------|------|
| **大型商场** | 数字室分 / 传统室分 | 容量需求大 |
| **写字楼** | 新型数字室分 | 高端场景，支持 [[VoNR]] |
| **地下停车场** | 传统室分 / 室外对打 | 成本优先 |
| **地铁** | 泄露电缆 / 数字室分 | 隧道专用方案 |
| **电梯** | 电梯井天线 | 随梯覆盖 |
