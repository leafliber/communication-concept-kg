### 英文全称

- Vehicle to Everything

### 中文释义

- 车联网 / 车用无线通信技术

### 解释

V2X 是使车辆能够与周围环境（其他车辆、道路基础设施、行人、云端平台）进行信息交换的通信技术，是实现自动驾驶和智能交通系统的关键技术。

### V2X 四大通信类型

| 类型 | 全称 | 说明 | 通信距离 |
|------|------|------|---------|
| **V2V** | Vehicle to Vehicle | 车间通信 | 300-1000 m |
| **V2I** | Vehicle to Infrastructure | 车与路侧单元通信 | 100-500 m |
| **V2P** | Vehicle to Pedestrian | 车与行人/弱势道路使用者通信 | 10-100 m |
| **V2N** | Vehicle to Network | 车与云端平台通信 | 广域 |

### V2X 两大技术路线对比

| 特性 | C-V2X (蜂窝车联网) | DSRC (专用短程通信) |
|------|------------------|---------------------|
| **组织** | 3GPP 主推 | IEEE 802.11p |
| **基于** | LTE / 5G 蜂窝网络 | Wi-Fi 衍生技术 |
| **优势** | 覆盖广、可演进、网络协调 | 标准化早、技术成熟 |
| **劣势** | 依赖网络覆盖 | 扩展性差、难以支持高级功能 |
| **5G V2X** | 满足全部自动驾驶场景需求 | 不支持 |

### C-V2X 演进路线

```
LTE V2X → LTE eV2X → 5G V2X (NR V2X)
(Rel-14)   (Rel-15)    (Rel-16)
```

| 版本 | 主要特性 | 时延 | 可靠性 |
|------|---------|------|--------|
| **LTE V2X** | 基本安全消息转发 | ~100 ms | 90% |
| **LTE eV2X** | 增强驾驶安全用例 | ~20 ms | 99% |
| **5G V2X (NR)** | 高级自动驾驶 | <1 ms | 99.999% |

### 5G V2X (NR V2X) 关键场景 (3GPP Rel-16)

| 场景 | 说明 | [[QoS]] 要求 |
|------|------|------------|
| **编队行驶** | 车辆自动编队协同行驶 | 超低时延、高可靠 |
| **传感器共享** | 车辆间共享摄像头/雷达感知数据 | 低时延、大带宽 |
| **远程驾驶** | 操作员远程接管驾驶 | <10 ms 时延 |
| **自动驾驶** | 车辆自主决策行驶 | <1 ms (URLLC) |

### V2X 典型应用场景

| 场景 | 说明 |
|------|------|
| **防碰撞预警** | 检测碰撞风险，提前预警 |
| **紧急制动预警** | 前车急刹时后车收到预警 |
| **弯道预警** | 弯道盲区来车提醒 |
| **绿波车速引导** | 交通信号灯配时优化通行 |
| **编队行驶** | 商用车队自动编队，降低风阻油耗 |
