### 英文全称

- Network Slice Selection Function

### 中文释义

- 网络切片选择功能

### 解释

NSSF 是 5GC（5G核心网）的核心网元之一，负责在用户（UE）接入网络时，根据用户的签约信息和业务需求，为用户选择合适的网络切片 (Network Slice)。

### NSSF 在切片选择中的位置

```
┌──────────────────────────────────────────────────────────────┐
│                         5GC                                  │
│                                                              │
│   ┌─────────┐                                               │
│   │  NSSF  │  ← 切片选择决策                                  │
│   └───┬─────┘                                               │
│       │                                                      │
│   ┌───┴──────────────┐  ┌─────────┐  ┌─────────┐            │
│   │     [[NRF]]       │←─│  [[AMF]]  │←─│  UE     │            │
│   │  (查询可用切片)    │  │(执行切片选择)│  │(请求接入)│            │
│   └──────────────────┘  └─────────┘  └─────────┘            │
└──────────────────────────────────────────────────────────────┘
```

### 切片选择流程

| 步骤 | 说明 |
|------|------|
| 1 | UE 在注册请求中携带 **NSSAI** (Network Slice Selection Assistance Information) |
| 2 | [[AMF]] 向 NSSF 查询用户的 Allowed NSSAI（允许的切片列表） |
| 3 | NSSF 根据 UE 签约信息 ([[UDM]]) 决定允许接入的切片 |
| 4 | NSSF 向 [[NRF]] 查询选定切片的 [[AMF]] 地址 |
| 5 | NSSF 返回 Allowed NSSAI 和目标 [[AMF]] 地址给当前 [[AMF]] |
| 6 | [[AMF]] 将 UE 注册到正确的切片 |

### S-NSSAI (Single NSSAI)

切片标识符结构：

| 字段 | 说明 | 示例 |
|------|------|------|
| **SST** (Slice/Service Type) | 切片业务类型 | 1=eMBB, 2=URLLC, 3=mMTC |
| **SD** (Slice Differentiator) | 切片差异化符 | 用于区分同一类型的多个切片 |

| SST 值 | 切片类型 | 说明 |
|--------|---------|------|
| 1 | eMBB | 增强移动宽带，大带宽业务 |
| 2 | URLLC | 超可靠低时延通信 |
| 3 | mMTC | 海量机器类通信 |
| 0 | 标准化切片 | 3GPP 定义的标准化切片 |

**示例**:
- `SST=1, SD=000001` → eMBB 切片 1
- `SST=2, SD=000001` → URLLC 切片 1
- `SST=3, SD=000001` → mMTC 切片 1

### NSSF 核心功能

| 功能 | 说明 |
|------|------|
| **Allowed NSSAI 确定** | 根据用户签约信息 ([[UDM]])，确定用户可接入的切片列表 |
| **Target AMF 选择** | 选择服务于目标切片的 [[AMF]] (可能跨 [[AMF]] Pool) |
| **切片重选择** | UE 移动或切片变更时，重新选择切片 |
| **配置管理** | 配置每个切片与 NF 的映射关系 |
| **切片间通信** | 支持 UE 在不同切片间切换 |

### 相关概念

- [[网络切片]] - NSSF 服务的对象
- [[AMF]] - NSSF 为 [[AMF]] 选择合适的切片
- [[NRF]] - NSSF 向 [[NRF]] 查询切片的可用 NF 实例
- [[5GC]] - NSSF 所在的 5G 核心网
- [[SBI]] - NSSF 对外提供服务化接口
