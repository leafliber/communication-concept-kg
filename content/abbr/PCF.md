### 英文全称

- Policy Control Function

### 中文释义

- 策略控制功能

### 解释

PCF 是 5GC（5G核心网）的核心网元，负责统一管理和下发用户的策略规则（包括 [[QoS]] 策略、计费策略、切片选择策略等），是 5G 网络实现差异化服务的核心控制点。

### PCF 在 5GC 架构中的位置

```
┌──────────────────────────────────────────────────────┐
│                      5GC                             │
│  ┌─────────┐        ┌─────────┐                      │
│  │   PCF   │←──────→│  [[NRF]]  │  (服务注册)           │
│  └────┬────┘        └─────────┘                      │
│       │                                              │
│  ┌────┴────┐  ┌────┴────┐  ┌─────────┐               │
│  │  [[AMF]]  │  │  [[SMF]]  │  │  [[AUSF]] │               │
│  │(策略获取)│  │(策略执行)│  │(认证协同)│               │
│  └─────────┘  └─────────┘  └─────────┘               │
└──────────────────────────────────────────────────────┘
```

### 核心功能

| 功能 | 说明 |
|------|------|
| **统一策略管理** | 集中管理 [[QoS]] 规则、计费规则 |
| **用户策略下发** | 向 [[AMF]] 下发接入层策略，向 [[SMF]] 下发会话层策略 |
| **切片策略控制** | 决定用户可接入的切片 ([[NSSF]] 协同) |
| **URLLC 保障** | 为 URLLC 业务预留资源，保障低时延 |
| **计费策略** | 决定用户如何计费 (后付费/预付费/套餐) |
| **SLA 保障** | 确保业务的 [[QoS]] 承诺 |

### PCF 与 PCRF 的区别

| 特性 | PCF (5GC) | PCRF (EPC) |
|------|-----------|------------|
| **架构** | 服务化 (NF)，基于 [[SBI]] | 单体化 |
| **切片感知** | 支持切片级策略控制 | 不支持网络切片 |
| **动态策略** | 可实时更新策略 | 预配置为主 |
| **URLLC 支持** | 原生支持 | 不支持 |
| **UPF 协同** | 通过 [[SMF]] 控制 [[UPF]] 的 [[QoS]] 执行 | 通过 P-GW 控制 |

### 策略类型

| 策略类型 | 说明 | 下发对象 |
|---------|------|---------|
| **接入层策略 (AM policy)** | 无线接入相关的策略 | [[AMF]] |
| **会话层策略 (SM policy)** | PDU Session 相关的策略 | [[SMF]] → [[UPF]] |
| **切片策略** | 切片选择和资源保障 | [[AMF]] + [[SMF]] |
| **URLLC 策略** | 超低时延业务保障 | [[SMF]] + [[UPF]] |

### 典型业务策略配置

| 业务 | PCF 策略配置 |
|------|-------------|
| **VoNR** | QCI=1, GBR, 优先保障 |
| **视频会议** | QCI=2, GBR, 优先调度 |
| **云游戏** | QCI=3, 低时延保障 |
| **普通上网** | QCI=8/9, Non-GBR |
| **URLLC 自动驾驶** | QCI=65/68, 极低时延保障 |

### 相关概念

- [[5GC]] - PCF 所在的 5G 核心网
- [[QoS]] - PCF 控制的核心
- [[SMF]] - 执行 PCF 下发的会话策略
- [[UPF]] - 执行 PCF 下发的用户面策略 ([[QoS]])
- [[NRF]] - PCF 的服务注册中心
- [[SBI]] - PCF 对外提供服务化接口

> 反向链接: [[SMF]] 从 PCF 获取会话策略后，指示 [[UPF]] 执行 [[QoS]] 规则，实现端到端的服务质量保障
