### 英文全称

- Narrow Band Internet of Things

### 中文释义

- 窄带物联网

### 解释

NB-IoT 是由 3GPP 标准化定义的蜂窝物联网技术，专为低功耗广覆盖（LPWA）场景设计，是 5G mMTC 的重要组成部分。

### 核心特点

| 特性 | 说明 |
|------|------|
| 覆盖增强 | 164dB MCL（比 GSM 提升 20dB） |
| 功耗 | PSM/eDRX 低功耗模式 |
| 成本 | 低于 5 美元 |
| 连接密度 | 每小区 10 万连接 |
| 带宽 | 180kHz（独立或内嵌） |

### 部署方式

| 模式 | 说明 |
|------|------|
| 独立部署 | 使用 200kHz 独立频段 |
| 保护带部署 | 利用 LTE 保护带 |
| 带内部署 | 占用 LTE 资源块 |

### 典型应用

- 智能抄表（水/电/气）
- 智慧城市基础设施
- 环境监测
- 资产追踪
- 智能停车
