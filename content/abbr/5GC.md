### 英文全称

- 5G Core

### 中文释义

- 5G核心网

### 解释

5GC 是 5G 独立组网（SA）的核心网，采用服务化架构（SBA），网元间通过 HTTP/2 接口通信。是 5G 时代的关键基础设施。

### 核心网元

| 网元 | 功能 |
|------|------|
| [[AMF]] | 接入和移动性管理 |
| [[SMF]] | 会话管理 |
| [[UPF]] | 用户面数据转发 |

### 与 EPC 对比

| 特性 | 5GC (SA) | EPC (NSA) |
|------|-----------|-----------|
| 架构 | SBA 服务化 | 传统电信架构 |
| 网元间接口 | HTTP/N4 | GTP |
| 网络切片 | 支持 | 不支持 |
| 部署方式 | 云原生/容器化 | 虚拟化 |

### 语音方案

- **VoNR**: 5G 直接承载语音（首选）
- **EPS Fallback**: 语音回落到 LTE/EPC

### 相关概念

- [[AMF]] - 接入管理功能
- [[SMF]] - 会话管理功能
- [[UPF]] - 用户面功能
- [[NSA]] - 非独立组网（使用 EPC）
- [[SA]] - 独立组网（使用 5GC）
- [[VoNR]] - 5G 语音
- [[IMS]] - IP 多媒体子系统
