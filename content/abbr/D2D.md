### 英文全称

- Device to Device

### 中文释义

- 设备直连通信 / 终端直通

### 解释

D2D 是使相邻的终端设备之间能够直接通信而不经过基站或核心网的技术，是 5G NR 的重要特性之一，可应用于公共安全、紧急通信、车联网等场景。

### D2D vs 传统通信

| 通信模式 | 路径 | 说明 |
|---------|------|------|
| **传统蜂窝** | UE → [[gNB]] → 核心网 → [[gNB]] → UE | 必须经过基站和核心网 |
| **D2D (直连)** | UE ↔ UE | 不经过基站，终端直接通信 |

### D2D 类型

| 类型 | 说明 | 标准 |
|------|------|------|
| **LTE D2D (ProSe)** | LTE Device-to-Device proximity services | 3GPP Rel-12/13 |
| **NR D2D (Sidelink)** | 5G NR Sidelink 直连 | 3GPP Rel-16/18 |

### D2D 典型应用场景

| 场景 | 说明 |
|------|------|
| **公共安全** | 基站损毁时，终端直连维持通信 |
| **应急通信** | 灾害现场无覆盖时，终端自组网 |
| **V2V ([[V2X]])** | 车与车直接通信，极低时延 (<10ms) |
| **物联网** | 智能家居设备本地直连，不耗基站资源 |
| **本地卸载** | 邻近设备间直接传输文件 |

### NR Sidelink 架构

```
┌─────────────────────────────────────────────────────────────┐
│                 NR Sidelink (D2D) 通信                       │
│                                                              │
│  ┌──────────────────────────────────────────────────────┐  │
│  │                    传统 5G NR 网络                      │  │
│  │  UE A ←──────→ [[gNB]] ←──────→ UE B                  │  │
│  └──────────────────────────────────────────────────────┘  │
│                           ↓                                  │
│  ┌──────────────────────────────────────────────────────┐  │
│  │                    Sidelink 直连                       │  │
│  │  UE A ←━━━━━━━━━━━━━━━━━━━━━━→ UE B                  │  │
│  │         (不经过 gNB，直接通信)                          │  │
│  └──────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```

### Sidelink 资源分配模式

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| **Mode 1 (基站调度)** | 基站分配 Sidelink 资源 | 有网络覆盖场景 |
| **Mode 2 (终端自管理)** | 终端自主选择 sidelink 资源 | 无网络覆盖（如公共安全） |

### D2D 与 [[V2X]] 的关系

| 维度 | LTE V2X | NR V2X (Sidelink) |
|------|---------|-------------------|
| **技术基础** | LTE D2D (ProSe) | NR Sidelink |
| **时延** | ~100 ms | <10 ms (增强) |
| **可靠性** | 99% | 99.999% |
| **支持场景** | 基本安全 | 自动驾驶等高级场景 |

### D2D 关键技术

| 技术 | 说明 |
|------|------|
| **发现机制** | 终端发现周围可通信的邻近设备 |
| **同步** | 无网络覆盖时，终端间同步 |
| **资源池** | Sidelink 专用资源池配置 |
| **功率控制** | D2D 发射功率控制，避免干扰 |

### 相关概念

- [[V2X]] - V2V 是 D2D 的重要应用
- [[gNB]] - 有网络覆盖时参与 D2D 资源调度
- [[URLLC]] - D2D 可实现极低时延
- [[NR]] - NR Sidelink 是 5G D2D 标准
