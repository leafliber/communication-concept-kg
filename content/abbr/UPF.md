### 英文全称

- User Plane Function

### 中文释义

- 用户面功能

### 解释

UPF 是 5G 核心网的用户面网元，负责数据包的转发、隧道封装、流量统计等。是 5G 网络中用户数据的实际处理节点。

### 核心功能

- **数据转发**: 数据包的路由和转发
- **隧道封装**: GTPU 隧道封装/解封装
- **流量统计**: 计费和性能统计
- **QoS 保障**: 执行 SMF 下发的 QoS 规则
- **数据包过滤**: 深度包检测（DPI）
- **边缘计算**: 支持 MEC（移动边缘计算）部署

### 与 SMF 配合

- SMF 为控制面，制定策略
- UPF 为用户面，执行转发
- N4 接口用于两者通信

### 部署位置

- 核心机房：集中式部署
- 边缘机房：分布式部署（低时延场景）
