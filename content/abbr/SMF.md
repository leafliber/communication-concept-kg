### 英文全称

- Session Management Function

### 中文释义

- 会话管理功能

### 解释

SMF 是 5G 核心网（5GC）的关键网元，负责用户会话的建立、管理和释放，以及 IP 地址分配、QoS 控制等。

### 核心功能

- **会话管理**: 建立、修改、释放 PDU 会话
- **地址分配**: 为 UE 分配 IP 地址（IPv4/IPv6）
- **QoS 控制**: 执行 QCI 和 ARP 管理
- **计费**: 收集会话相关的计费信息
- **下行数据通知**: 触发 UPF 向 UE 发送下行数据

### 与 UPF 配合

- SMF 为控制面，负责策略控制
- UPF 为用户面，负责数据转发
- 两者通过 N4 接口通信
